# ushell-module-rbenv
> Unix like shell module for rbenv.


## install-on-linux:
- ./install.sh

## install-on-mac:
- ./install-on-mac.sh

## rbenv init:
```bash
if which rbenv > /dev/null; then 
    eval "$(rbenv init -)"; 
fi
```
